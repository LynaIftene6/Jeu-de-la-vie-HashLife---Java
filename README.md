# Jeu-de-la-vie-HashLife---Java

## Description

Implémentation du Jeu de la vie de Conway avec optimisation via l’algorithme HashLife, permettant d’accélérer considérablement les simulations.

---

## Fonctionnalités

* Interface graphique (Java Swing)
* Zoom et déplacement dans la grille
* Simulation optimisée (HashLife)
* Multithreading pour de meilleures performances
* Benchmark intégré

---

## Technologies

* Java
* Java Swing
* Multithreading

---

## Lancer le projet

```bash
git clone https://github.com/LynaIftene6/Jeu-de-la-vie-HashLife---Java.git
cd Jeu-de-la-vie-HashLife---Java
javac -d build src/algorithms/*.java src/core/*.java src/rules/*.java src/ui/*.java 
java -cp build ui.Main
```
